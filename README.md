## 📂 Project Structure

The project has been organized into logical subdirectories for better navigation:

### 1. 🧱 Basics

Foundational Python concepts and data structure manipulations.

- `basics/list_basics.py`: Exploration of list indexing, appending, and basic methods with detailed comments.
- `basics/list_operations.py`: Practicing list iteration, measuring length, and modifying content.

### 2. ⚡ Algorithms

Implementations of classic computer science algorithms, now including **Big O Notation** analysis.

#### 🔍 Searching

- `algorithms/searching/linear_search.py`: Simple linear search with **O(n)** time complexity.
- `algorithms/searching/binary_search.py`: Efficient binary search on sorted lists with **O(log n)** time complexity.

#### 🧼 Sorting

- `algorithms/sorting/bubble_sort.py`: Implementation of the Bubble Sort algorithm, demonstrating both **Ascending** and **Descending** order sorts. **O(n²)**

#### 🔄 Recursion

- `algorithms/recursion/recursion_examples.py`: Comprehensive examples of recursive functions: Sum, Factorial, and Fibonacci.
- `algorithms/recursion/fibonacci.py`: Comparative file showing both **Iterative** and **Recursive** Fibonacci approaches.

## 🛠️ Getting Started

### Prerequisites

- Python 3.x installed on your machine.

### Running the Scripts

Navigate to the desired subdirectory and run scripts using the Python interpreter:

```powershell
# Example: Running the Bubble Sort algorithm
python algorithms/sorting/bubble_sort.py
```
