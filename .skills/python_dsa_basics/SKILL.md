## Instructions for Antigravity

When working with this skill, ensure that all implementations follow these principles:

1. **Clear Comments**: Provide line-by-line comments for algorithm logic.
2. **Big O Notation**: Always mention the time and space complexity at the top of the file.
3. **Structured Folders**: Keep algorithms in their respective categories (e.g., `searching`, `sorting`, `recursion`).
4. **Test Cases**: Include a simple testing block at the bottom of each file.
5. **Readable Var Names**: Use descriptive variable names (e.g., `num_count` instead of `nc`).
